Tag Selector

Tag selectors apply styles to all elements of a specific HTML tag type (e.g., all paragraphs or headers).

This paragraph is styled using a tag selector, which has the lowest priority.

Class Selector

Class selectors apply styles to elements with a specific class attribute. They have a higher priority than tag selectors.

This paragraph is styled using a class selector, which has a higher priority than tag selectors.

ID Selector

ID selectors apply styles to elements with a specific ID attribute. They have a higher priority than both tag and class selectors.

This paragraph is styled using an ID selector, which has a higher priority than class and tag selectors.

Inline Style

Inline styles are applied directly to an element using the style attribute. They have a higher priority than selectors defined in the stylesheet.

This paragraph is styled using an inline style, which has a higher priority than selectors in the CSS file.

Using !important

The !important rule is used to override all other styles, including inline styles. It should be used sparingly as it makes code less flexible.

This paragraph is styled using the !important rule, which overrides all other styles.
