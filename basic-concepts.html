Introduction to CSS

CSS (Cascading Style Sheets) is a styling language used to control the appearance of HTML elements. CSS can be applied to HTML tags in three ways:

  • Inline styles: applied directly to an HTML element using the style attribute.
  • Internal styles: placed within a <style> tag in the HTML file.
  • External styles: linked via an external CSS file placed within a <link> tag in the HTML file.

Basic Syntax of CSS

A CSS rule consists of a selector and a declaration block. Example:

h1 {
  color: blue;
  font-size: 24px;
}

The selector targets an element (in this case, h1), and the properties (such as color and font-size) change specific aspects of that element.

Types of Selectors

Selectors allow us to target specific elements:

  • Tag selectors: Apply styles to all elements of a certain type (e.g., h1, p).
  • Class selectors: Target elements with a specific class, marked with a dot (.).
  • ID selectors: Target elements with a specific ID, marked with a hash symbol (#).
