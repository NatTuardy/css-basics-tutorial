Border-box and Content-box

CSS uses different models to define the dimensions of an element. The two main models are content-box and border-box.

content-box (default): Width and height are defined only for the content area, while padding and border are added outside of these dimensions.

border-box: Width and height include content, padding, and border.

Content Box: width 200px + padding + border
Border Box: width 200px includes padding and border

Display: Block, Inline, Inline-block

The display property defines how an element is displayed on the page.

Block: Element takes up the full width available and starts on a new line.

Inline: Element only takes up as much width as needed and does not break the text flow.

Inline-block: Element takes up only necessary width but behaves like a block, allowing you to set dimensions.

Margin Collapsing

When two vertical margins meet, they collapse into one margin, which is equal to the larger of the two (not their sum).

Nested Block Positioning

To position an element inside another, you can use margin, padding, or positioning properties such as relative and absolute.

Vertical Align

The vertical-align property works with inline or inline-block elements and allows them to align vertically relative to the baseline.

Background Properties

The background properties allow you to set and control the appearance of background images.

background-repeat: Defines whether the background image repeats. Values: repeat, no-repeat, repeat-x, repeat-y.

background-position: Sets the position of the background image (e.g., center, top right).

background-size: Defines the size of the background image (e.g., cover, contain).

background-attachment: Fixes the background relative to the page. Values: local, fixed, scroll.
